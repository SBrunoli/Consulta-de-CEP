## 💻 Sobre o Projeto

Este projeto é uma ferramenta web intuitiva e rápida para **Consulta de Endereços** através do CEP, consumindo a API pública do **ViaCEP**.

O grande diferencial deste projeto é que ele foi **idealizado e desenvolvido 100% por mim**, começando pelo desenho da interface (UI/UX) e seguindo para a implementação completa do código front-end. O objetivo foi criar uma experiência de usuário fluida, bonita e livre de erros.

## ✨ Funcionalidades

✅ **Busca Instantânea:** Retorna Logradouro, Bairro, Cidade, Estado, UF e Região ao digitar o CEP.

✅ **Validação Inteligente:** Valida o formato do CEP (8 números) antes de enviar a requisição, prevenindo erros.

✅ **Prevenção de Erros na Entrada:** O campo de texto aceita apenas números e possui limite de caracteres.

✅ **Tratamento de Exceções:** Feedback visual claro para CEPs não encontrados ou falhas de conexão.

✅ **Copiar para Clipboard (UX+):** Clique em qualquer dado do endereço para copiá-lo instantaneamente.

✅ **Feedback Visual de Cópia:** Um tooltip animado ("Copiado!") aparece ao clicar nos dados.
